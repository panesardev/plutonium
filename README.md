# Plutonium

> Reactivity focused coding tutorials

A modern Angular-based platform dedicated to teaching reactive and declarative programming using RxJS and Signals. Learn best practices for building reactive applications with Angular's latest features.

![Plutonium](https://plutoniumx.vercel.app/icons/plutonium.png)

## 🌐 Live 

**[plutoniumx.vercel.app](https://plutoniumx.vercel.app)**

## 📖 About

Plutonium is an educational platform focused on teaching developers how to build reactive and declarative applications using Angular. The platform provides in-depth articles and tutorials covering modern Angular features, including:

## 🚀 Features

- **Article Library**: Comprehensive tutorials on reactive programming
- **Hashtag System**: Easy content discovery and organization
- **User Authentication**: Login functionality for personalized experience
- **Search Functionality**: Find relevant content quickly
- **Responsive Design**: Optimized for all devices
- **Modern UI**: Clean, professional interface built with Tailwind CSS

## 🛠️ Tech Stack

- **Angular** - Modern web application framework
- **Tailwind** - Utility-first CSS framework
- **Firebase** User authentication and database
- **ngx-markdown** - Content and markdown management
- **Vercel** - Hosting and continuous deployment
